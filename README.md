研究内容  
居民用户需要实现的目标是：  
实现居民用户可以进行账号注册、系统登录、查看个人信息、修改个人信息、查看活动信息、参加活动报名信息、查看公告信息、留言反馈、提交报修申请、物业费缴纳等功能。  
管理员需要实现的目标是：  
管理员包括居民信息管理、活动管理、公告信息管理、投诉反馈管理、维修报修管理、物业费管理等功能。
